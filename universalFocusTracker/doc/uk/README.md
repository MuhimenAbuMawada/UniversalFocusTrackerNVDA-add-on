# Універсальний відстежувач фокусу для NVDA

**Універсальний відстежувач фокусу** — це професійний додаток до NVDA, розроблений для синхронізації курсору миші з фокусом NVDA та навігатором об'єктів. Він забезпечує ідеальне позиціонування миші на активному елементі, що дарує плавний досвід навігації у різних застосунках і сучасних веб-браузерах.

---

## Ключові можливості

* **Точне вирівнювання курсору:** Автоматично переміщує мишу в геометричний центр фокусованого об'єкта або курсору навігатора.
* **Повна інтеграція з браузерами:** Розширене відстеження для Brave, Chrome і Firefox, щоб миша слідувала за курсором читання навіть у складних веб-документах.
* **Інтелектуальна обробка меню:** Автоматично призупиняє відстеження під час роботи з меню, щоб уникнути випадкового закриття чи втрати фокусу.

---

## Налаштування та конфігурація

Додаток повністю інтегровано в налаштування NVDA. Користувачі можуть керувати поведінкою відстеження через спеціальну категорію налаштувань:

1.  Відкрийте **меню NVDA** (NVDA+N).
2.  Перейдіть до **Параметри** -> **Налаштування**.
3.  Виберіть **Універсальний відстежувач фокусу**.
4.  **Перемикач увімкнення/вимкнення:** Використовуйте відповідний прапорець, щоб миттєво увімкнути або вимкнути відстеження миші.
