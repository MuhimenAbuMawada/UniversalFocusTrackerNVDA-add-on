# Universal Focus Tracker (Универсальное отслеживание фокуса)

Дополнение Universal Focus Tracker автоматически перемещает указатель мыши в центр текущего элемента, находящегося под фокусом NVDA.

## Основные возможности

Дополнение отслеживает изменение фокуса и перемещение курсора ввода.
При переключении между элементами мышь автоматически следует за объектом, на котором находится виртуальный курсор NVDA.
Это особенно полезно для пользователей, которые используют мышь в паре со скринридером.
Кроме того, дополнение полезно для людей, которые проводят демонстрации экрана, когда ученикам нужно видеть, где находится текущий фокус с помощью указателя мыши, а также для пользователей, у которых на компьютерах нет клавиши контекстного меню («Applications»), и они привыкли использовать правую кнопку мыши естественным образом.

## Поведение в различных приложениях

В обычных приложениях (текстовые редакторы, диалоговые окна, панели управления) дополнение перемещает мышь мгновенно при получении элементом фокуса.

В браузерах и других приложениях с режимом обзора (browse mode) используется специальный механизм.
Дополнение запускает таймер с интервалом 100 миллисекунд, который отслеживает положение навигационного объекта NVDA.
Это обеспечивает корректное отслеживание мыши при перемещении по веб-страницам с помощью клавиш навигации.

## Исключения

Дополнение не перемещает мышь при фокусе на следующих элементах:
• Всплывающие меню (popup menu)
• Панели меню (menu bar)
• Меню (menu)
• Пункты меню (menu item)

Это сделано для предотвращения нежелательного поведения при работе с выпадающими списками и контекстными меню.

## Настройки

Дополнение добавляет панель настроек в диалог параметров NVDA.
Для доступа к настройкам выполните следующие действия:
Откройте меню NVDA.
Выберите пункт «Параметры».
Нажмите «Настройки NVDA».
В списке категорий найдите «Отслеживание фокуса».

В панели настроек доступна одна опция:
«Включить отслеживание» — флажок, который включает или отключает автоматическое перемещение мыши.
По умолчанию отслеживание включено.

## Использование

После установки дополнения мышь будет автоматически следовать за фокусом NVDA.
При переключении между окнами, кнопками, полями ввода и другими элементами указатель мыши перемещается в центр каждого элемента.
При перемещении курсора ввода в текстовых полях мышь также следует за курсором.
При работе в браузере мышь перемещается при навигации по ссылкам, заголовкам и другим элементам страницы.

## Отключение функции

Если вам нужно временно отключить отслеживание мыши, снимите флажок «Включить отслеживание»в настройках NVDA.
После снятия флажка мышь перестанет автоматически перемещаться до повторного включения функции.
